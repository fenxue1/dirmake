# 分页式界面交互说明

本软件 GUI 已重构为分页模式，包含以下页签：

- 提取CSV：选择项目目录、配置提取参数，输出 `ty_text_out.csv`
- 从CSV生成C：选择 CSV 输入与 C 输出文件，配置生成参数
- 设置：说明与全局配置入口（当前版本内嵌说明）
- 日志：显示运行过程输出与提示

主要交互与特性：

- 页签切换：顶部 QTabWidget，点击切换；支持快捷键与滑动
- 快捷键：`Ctrl+Tab` 下一页；`Ctrl+Shift+Tab` 上一页
- 面包屑指示：窗口顶部显示当前位置（例如：`首页 / 提取CSV`）
- 懒加载：首次切换到相关页时构建界面，减少初始加载成本
- 状态持久化：退出并重启后自动恢复到上次打开的页签
- 触控/滑动：支持鼠标或触屏左右滑动快速切换页签（水平位移>60px，<500ms）
- 拖拽：拖入目录自动设置根目录与项目目录并触发“一键运行”；拖入 CSV 自动进入从 CSV 生成 C 流程

使用模式：

- 仅提取CSV：在“提取CSV”页配置参数，点击“从代码提取到CSV”
- 仅生成C：在“从CSV生成C”页选择输入与输出，点击“生成”
- 提取并生成：在“从CSV生成C”页选择“提取并生成”，点击“生成”，将依次执行提取与生成

注意事项：

- Windows 7 兼容：使用 PyQt5 5.15.x 构建打包；若缺少 Qt 依赖，请先安装再运行
- 性能：懒加载减少初始渲染；切换动画为淡入效果（约 200ms）
- 文件路径：未显式设置时，会根据项目目录自动填充常见默认路径

问题反馈：

- 若运行失败，请查看“日志”页输出并将 `dist_win7\build_x64\warn-main.txt` 反馈
